# Modelo_regresion
## Regresión lineal simple

Es un modelo estadístico que analiza la relación entre dos variables cuantitativas 
continuas para predecir el valor de una variable dependiente (respuesta) en función 
de una única variable independiente (predictora). 
Su objetivo principal es determinar cómo cambia la variable de respuesta a medida 
que varía la predictora, permitiendo generar pronósticos y analizar tendencias en 
disciplinas como la economía, medicina y sociología. Además, muchos métodos de 
ciencia de datos, como el machine learning y la inteligencia artificial, utilizan la 
regresión lineal para resolver problemas complejos. 
En esencia, una técnica de regresión lineal simple intenta trazar un gráfico lineal entre 
dos variables de datos, x e y. Como variable independiente, x se traza a lo largo del 
eje horizontal. Las variables independientes también se denominan variables 
explicativas o variables predictivas. La variable dependiente, y, se traza en el eje 
vertical. También puede hacer referencia a los valores y como variables de respuesta 
o variables pronosticadas.
